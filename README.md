# Web-apps-html-css-js
A repository of beginner-friendly web applications built to practice core front-end development concepts such as UI design, logic handling, and interactivity.

#LoginPage

I developed a modern and responsive Login & Registration user interface using HTML, CSS, and JavaScript. This project features a clean design with smooth transitions between login and signup forms, providing a seamless user experience. The interface includes an interactive popup modal that allows users to switch between login and registration without reloading the page. It uses CSS animations and JavaScript event handling to create a dynamic and visually appealing flow.
